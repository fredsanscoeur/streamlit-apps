# Stock Market Visualizer（股票市场可视化器）

一个基于 Streamlit + Akshare 的中国股票市场数据可视化与分析工具。
提供交互式 K 线（Plotly）及多项分析功能：MA、RSI、布林带、筹码分布（估算）、组合跟踪、相关性分析、财务比率、基础预测与图表导出。
